# Python Scripts

这个文件夹包含所有的 Python 脚本。

## 脚本列表

### 金融数据相关
- `get_fred_data.py` - 从 FRED 获取经济数据
- `get_free_financial_data.py` - 获取免费金融数据
- `financial_data.py` - 金融数据处理
- `financial_data_qq.py` - 金融数据 QQ 发送
- `financial_data_tavily.py` - 使用 Tavily 获取金融数据
- `global_interest_rates.py` - 全球利率数据
- `fed_data_tavily.py` - 美联储数据（Tavily）
- `test_symbols.py` - 测试股票代码

### 利率分析相关
- `create_interest_rates_db.py` - 创建利率数据库
- `analyze_interest_rates.py` - 分析利率数据

### 黄金油价相关
- `get_gold_oil_prices.py` - 获取黄金和石油价格
- `gold_oil_prices.py` - 黄金石油价格处理
- `gold_oil_prices_alt.py` - 黄金石油价格（备用版本）

### LLM 论文相关
- `llm_paper_digest.py` - LLM 论文摘要
- `send_llm_paper_qq.py` - 发送 LLM 论文到 QQ

### 其他
- `email_processor.py` - 邮件处理

## 使用方法

运行脚本：
```bash
python python-scripts/脚本名.py
```
