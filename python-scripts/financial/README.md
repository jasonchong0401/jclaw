# 金融数据脚本

## 脚本列表

### 利率相关
- `analyze_interest_rates.py` - 分析利率数据
- `create_interest_rates_db.py` - 创建利率数据库
- `global_interest_rates.py` - 全球利率数据

### 经济数据获取
- `get_fred_data.py` - 从 FRED 获取经济数据
- `get_free_financial_data.py` - 获取免费金融数据
- `fed_data_tavily.py` - 使用 Tavily 获取美联储数据

### 金融数据
- `financial_data.py` - 金融数据处理
- `financial_data_qq.py` - 金融数据 QQ 发送
- `financial_data_tavily.py` - 使用 Tavily 获取金融数据

## 使用方法

```bash
cd python-scripts/financial
python 脚本名.py
```
